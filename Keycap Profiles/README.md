# Keycap Profiles — Reference Documentation

> Profile specifications and descriptions for keycap sets used across Keychron keyboards.

Part of the [Keychron Keyboards Hardware Design](https://github.com/Keychron/Keychron-Keyboards-Hardware-Design) repository.

## Available Profiles

| Profile | Description |
|---|---|
| [Cherry](./Cherry%20Profile/README.md) | Industry-standard low-profile sculpted keycap shape |
| [KSA](./KSA%20Profile/README.md) | Keychron Spherical Angled — tall spherical sculpted profile |
| [LSA](./LSA%20Profile/README.md) | Low-profile Spherical Angled — compact spherical sculpted profile |
| [MDA](./MDA%20Profile/README.md) | Mid-height uniform profile with gentle sculpt |
| [OEM](./OEM%20Profile/README.md) | Standard tall cylindrical profile common on prebuilt boards |
| [OSA](./OSA%20Profile/README.md) | OEM Spherical Angled — tall spherical sculpted profile |

## Note

These directories currently contain profile descriptions only. Keycap CAD geometry files (STEP) are included within individual keyboard model directories where applicable (e.g., `Q-Series/Q11/Q11-OSA-Keycap-20221212.stp`).
