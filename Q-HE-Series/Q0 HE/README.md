# Keychron Q0 HE — Hardware Design Files

> Hall Effect numpad files for the Q0 HE, including enclosure parts, knob, plate files, keycap, stabilizer, and a complete assembled model.

**Product page:** [Keychron Q0 HE](https://www.keychron.com/products/keychron-q0-he-qmk-wireless-custom-number-pad)

![Keychron Q0 HE](https://www.keychron.com/cdn/shop/files/Keychron_Q0_HE_QMK_Wireless_Custom_Number_Pad-1.jpg?v=1768544284&width=3200)

## Specifications

| Spec | Detail |
|---|---|
| Layout | Numpad |
| Knob | Yes |
| Switch Type | Magnetic / Hall Effect |
| Connectivity | 2.4 GHz / Bluetooth / USB-C |
| Firmware | Keychron Launcher / QMK |

## Included Files

| File | Description |
|---|---|
| `Q0_HE_Bottom_Case.stp` | Bottom case enclosure |
| `Q0_HE_Full_Model_20260411.stp` | Complete assembled full model |
| `Q0_HE_Knob.stp` | Rotary encoder knob model |
| `Q0_HE_OSA_Keycap.stp` | OSA keycap model |
| `Q0_HE_Plate.dwg` | Plate drawing |
| `Q0_HE_Plate.dxf` | Plate geometry for CAD/CAM workflows |
| `Q0_HE_Plate.pdf` | Plate reference sheet |
| `Q0_HE_Top_Case.stp` | Top case enclosure |
| `Stabilizer-2.25u.stp` | 2.25u stabilizer model |

## Opening the Files

| Format | Free Software | Commercial Software |
|---|---|---|
| `.stp` | FreeCAD, Onshape | Fusion 360, SolidWorks |
| `.dwg` | LibreCAD (limited), QCAD | AutoCAD, DraftSight |
| `.dxf` | LibreCAD, QCAD | AutoCAD, DraftSight |
| `.pdf` | Any PDF viewer | — |
