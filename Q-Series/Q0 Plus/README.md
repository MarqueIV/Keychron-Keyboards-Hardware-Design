# Keychron Q0 Plus

> **Product page:** [Keychron Q0 Plus QMK Custom Number Pad](https://www.keychron.com/products/keychron-q0-plus-qmk-custom-number-pad)

A full-size number pad custom mechanical keyboard with all-metal body and programmable layout.

## Overview

The Q0 Plus is a 27-key gasket-mounted aluminum numpad designed for productivity workflows, data entry, and as a companion to compact keyboard layouts. Built with Keychron's signature CNC-machined aluminum case and gasket mounting system for a premium typing experience.

## Specifications

- **Layout:** Number pad, 27-key
- **Mount:** Gasket mount
- **Case:** CNC-machined aluminum
- **Connectivity:** USB-C wired
- **Hot-swap:** Yes (supports 3-pin and 5-pin switches)
- **Firmware:** QMK / VIA supported
- **Backlight:** South-facing RGB LED
