# Keychron M2 — Hardware Design Files

> Ultra-light wireless optical mouse with ergonomic shaping, tri-mode connectivity, and up to 8K polling depending on version.

**Product page:** [Keychron M2 Wireless Mouse](https://www.keychron.com/products/keychron-m2-wireless-mouse)

## Specifications

| Spec | Detail |
|---|---|
| Type | Wireless optical mouse |
| Shape | Ergonomic right-handed |
| Weight | 55 g |
| Sensor | PixArt 3395 / PixArt 3950 (version-dependent) |
| DPI | Up to 30000 |
| Polling Rate | Up to 8000 Hz (2.4 GHz / wired), 125 Hz (Bluetooth) |
| Connectivity | Bluetooth / 2.4 GHz / USB-C wired |
| Switch | Huano 80M micro switch |
| Battery | 600 mAh |
| Body Material | ABS |
